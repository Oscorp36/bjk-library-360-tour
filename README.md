# Billie Jean King Main Library – 360° Tour

A self-hosted 360° tour built with [Marzipano](https://www.marzipano.net/).
Hosted via GitHub Pages.

**Live tour:** https://YOUR-USERNAME.github.io/bjk-library-360-tour/

## About
This tour documents key spaces at the Billie Jean King Main Library (Long Beach, CA).
Captured with a Ricoh THETA SC2; assembled with Marzipano Tool; deployed on GitHub Pages.

## Navigation
- Click on arrow hotspots to move between scenes (Street View–style).
- Info hotspots link to relevant LBPL pages (services, hours, makerspace, etc.).

## Tech Notes
- Image resolution: 5376×2688 JPG (Ricoh THETA SC2)
- Export: Marzipano Tool (default settings)
- Hosting: GitHub Pages (`index.html` at repo root, `.nojekyll` present)
